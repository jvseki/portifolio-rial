# 🚀 Portfólio Profissional - Matheus Rial

Este é um projeto de site portfólio desenvolvido para **Matheus Rial**, estudante de Publicidade e Propaganda no Unisalesiano de Araçatuba. O objetivo do site é centralizar as informações acadêmicas, profissionais e facilitar o contato para orçamentos de sua agência de publicidade.

## 📱 Sobre o Projeto

O site foi desenhado com foco em **Mobile First**, garantindo uma experiência impecável para usuários que acessam através da bio do Instagram. O design segue uma estética moderna e minimalista, utilizando tons de preto e branco para ressaltar a identidade visual do cliente.

### ✨ Funcionalidades
* **Seção Sobre:** Detalhes da formação acadêmica (SESI 2023) e atual graduação (5º período na Unisalesiano).
* **Call to Action (CTA):** Botão direto para WhatsApp com mensagem personalizada para orçamentos.
* **Social Links:** Integração com Instagram profissional (@mths.publi) e pessoal.
* **Design Responsivo:** Adaptável para Celulares, Tablets e Desktops.
* **Animações:** Efeitos de *fade-in* ao rolar a página para uma navegação fluida.

---

## 🛠️ Tecnologias Utilizadas

O projeto utiliza o que há de mais moderno e eficiente para aplicações web leves:

* **Backend:** [Python](https://www.python.org/) com o micro-framework [Flask](https://flask.palletsprojects.com/).
* **Frontend:** HTML5, CSS3 (Custom Properties e Flexbox).
* **Interatividade:** JavaScript Vanilla (Intersection Observer API para animações).
* **Ícones:** [FontAwesome 6](https://fontawesome.com/).
